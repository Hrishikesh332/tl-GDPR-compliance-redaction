# Video Redaction (Frontend)

The app lives in the **frontend** folder.

## Structure

```
frontend/
├── package.json
├── index.html
├── vite.config.ts
├── tailwind.config.js  
├── tsconfig.json
├── src/                 
│   ├── index.css         
│   ├── main.tsx
│   ├── App.tsx
│   ├── components/
│   └── pages/
└── strand/               
    ├── tailwind-preset.js
    ├── tokens/
    ├── css/              
    ├── components/       
    ├── assets/           
    └── icons/
```

## Run the app

```bash
cd frontend && npm install && npm run dev
```

## Build

```bash
cd frontend && npm run build
```
